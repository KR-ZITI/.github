## 프로젝트 소개

### 서비스 이름 : 🤖 ZITI (Zealously Inspiring Together, Infinitely) 🤖

> 함께 무한히 성장하며 나아가는 나의 친구라는 의미를 담은 AI 서비스 

- **부제**: 사용자마다 공부(주식,언어,코딩), 친구, 선생님이 될 수 있는 소통형 서비스

- **타겟층**: 나를 항상 공감해주는 사람이 있으면 좋겠는 분들, 무언가를 배우고 싶거나 고민을 들어줄 사람이 필요한 분들
